# Limitr Demo
Limitr example server + web app. Shows a hybrid pricing policy connected to Limitr Cloud.

## Server
Three endpoints for changing the quantity of seats, AI chat tokens, and storage for one of our users.

### Run
```bash
cd server && npm install && npm run start
```

## Client
Set current user/customer, modify seats, AI chat tokens, and storage, and see the current usage/plan information from the open-source Limitr UI component.

### Run
```bash
cd client && npm install && npm run dev
```
